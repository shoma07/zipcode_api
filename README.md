# ZipcodeAPI

Zipcode search API

* Ruby version: 3.0.1

## Usage

```
$ RAILS_ENV=production bundle install
$ RAILS_ENV=production bin/rails db:migrate
$ RAILS_ENV=production bin/rails db:seed
```

## Docker

```sh
$ docker compose build
$ docker compose up
```
